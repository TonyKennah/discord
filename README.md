# Discord

A Java-based Discord application/bot project.

## Overview

This repository contains a Discord-related implementation written in Java.

## Technology Stack

- **Language**: Java

## Getting Started

### Prerequisites

- Java Development Kit (JDK) installed
- Maven or Gradle for dependency management

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/TonyKennah/discord.git
   cd discord
   ```

2. Build the project:
   ```bash
   mvn clean install
   ```
   or
   ```bash
   gradle build
   ```
